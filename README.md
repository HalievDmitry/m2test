Fisha_Test
===================
Our test module



-------------------------------
Requirements
------------
- Magento >= 2.1.9
- PHP >= 7.1.0


Installation by Composer
---------------------------------------------

    composer require fisha/test dev-master 
    bin/magento setup:upgrade
  
Installation by Git
---------------------------------------------
    
    git clone git@github.com:HalievDmitry/m2test.git app/code/Fisha/Test
